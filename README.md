💻 Personal Portfolio - Pablo Theves

This repository contains the source code for my personal portfolio website. It is designed to showcase my skills, professional experience, and software development projects, with a focus on Java and Web development.

🚀 Tech Stack
The project was built using a modern web development stack:

React 18

Vite (Fast build tool and HMR)

Tailwind CSS (Utility-first styling)

React Icons

JavaScript (ES6+)

🛠️ Key Features
Responsive Design: Fully optimized for mobile, tablet, and desktop devices using Tailwind's grid and flexbox systems.

Smooth Navigation: Sidebar and top-bar navigation (depending on screen size) with smooth scrolling between sections.

Dynamic Project Showcase: A grid layout displaying key projects with descriptions and direct links to GitHub repositories.

Contact Form: A functional user interface for inquiries, including name, email, and message fields.

🔧 How to Run Locally
Clone the repository:

Bash
git clone https://github.com/pablotheves/pablothevesportfolio.git
Install dependencies:

Bash
npm install
Start the development server:

Bash
npm run dev
View in browser: Open http://localhost:5173.
